PeopleSystem

Pequeno projeto utilizando os 4 pilares da programação orientada a objetos (POO).

A Classe Pessoa, dentro do pacote pessoa, é composta por 2 outras classes (Contato e Endereco), sendo assim
estou usando composição de classes na classe Pessoa.

A Classe Pessoa, é uma Super classe das classes (Funcionario e Cliente).
Sendo assim,usando herança as classes Funcionario e Cliente, herdam propriedades e metodos da classe Pessoa
